# Nightreign 攻略素材库

本目录用于存放从网上收集的参考攻略、图片和视频素材，按主题分类管理。

## 目录结构

```
resources/
├── bosses/        # Boss 攻略参考（打法、阶段分析、弱点等）
├── builds/        # 角色构建 & Tier List 参考
├── items/         # 物品/武器/装备相关素材
├── beginner/      # 新手入门攻略参考
├── walkthrough/   # 流程攻略参考
├── tips/          # 技巧与窍门
├── images/        # 通用图片素材（截图、地图、UI 等）
└── videos/        # 视频素材链接或本地文件
```

## 使用说明

- 每个主题目录下可创建子目录进一步细分（如 `bosses/heolstor/`）
- 图片建议使用 `.png` 或 `.webp` 格式
- 视频素材较大时，建议只保存链接（在 `.md` 文件中记录 URL）
- 文件命名使用英文小写 + 短横线，如 `heolstor-phase2-attack.png`
